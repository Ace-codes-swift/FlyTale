Flytale is a Hytale mod inspired by VS: Clockwork, designed to expand mobility and exploration in the game. With Flytale, players can glide, soar, and navigate Hytale's world like never before.

## Features

- **Aerial Mobility**: Glide and fly across the landscape with ease.
- **Smooth Mechanics**: Physics and controls optimized for fun and responsive flight.
- **VS: Clockwork-Inspired Design**: A familiar feel for players who loved VS: Clockwork's mechanics.

## Notes

- Flytale is compatible with other mods that do not modify core player movement.
- Early Access versions may require updates if the game's API changes.
